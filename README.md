# Go Docker App

This is a simple Go web application containerized using Docker.

## How to Run

1. Build the Docker image:
   ```bash
   docker build -t go-docker-app .
   ```
2. Run the container:
   ```bash
   docker run -p 8080:8080 go-docker-app
   ```
